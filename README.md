# llanquihue-tourS8

Proyecto: Servicios turísticos de la empresa Llanquihue Tour; actividad correspondiente a la semana 8, aplicación de herencia, sobrescritura y polimorfismo.
El proyecto busca representar distintos servicios turísticos, implementando una jerarquía de clases, permitiendo administrar rutas gastronómicas, paseos lacustres y excursiones culturales.

Estructura:
src/
└── main/
    └── java/
        └── cl/
            └── llanquihuetour/
                │
                ├── model/
                │   ├── ServicioTuristico.java
                │   ├── RutaGastronomica.java
                │   ├── PaseoLacustre.java
                │   └── ExcursionCultural.java
                │
                ├── data/
                │   └── GestorServicios.java
                │
                └── ui/
                    └── Main.java

Project Files/
└── pom.xml

Para ejecutar abrir el proyecto como proyecto Maven respetando la estructura indicada anteriormente.
Realizar un clean and build del proyecto.
Ejecutar la clase Main.java
También se puede ejecutar el archivo generado mediante terminal, utilizando  "java -jar .\target\llanquihueTourApp-1.0-SNAPSHOT.jar".
